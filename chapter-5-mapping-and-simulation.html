Chapter 5: Mapping and Simulation

Remember the "Process" tab? Time to run the next step! Make sure both "Map trace" and "Verilog simulation file" are ticked for this part, then run the process by double-clicking it. Your output should display "completed successfully", alongside green checkmarks.

Next up, let's set up the simulation. In the top bar, select the icon below, or go to "Tools > Simulation wizard".

Give it a name, then, once you reach the "process stage", select "post-map gate-level". This will generate a more like-life simulation of the circuit than plain RTL.
